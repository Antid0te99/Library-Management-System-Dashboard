# 📚 Library Management System Dashboard – Power BI

## 📌 Project Overview
This project is an interactive **Business Intelligence Dashboard** developed using **Power BI** for analyzing and monitoring library operations across multiple cities in Nepal. The dashboard provides insights into borrowing trends, member activity, overdue analysis, category popularity, and overall library performance.

The dataset used in this project was **created and prepared by myself** for academic purposes as part of the **Business Intelligence (BI)** course project in BIM 8th Semester.

The dashboard transforms raw transactional data into meaningful visual insights that support better decision-making for library management.

---

# 🎯 Objectives
The main objectives of this project are to:

- Monitor overall library performance using KPIs
- Analyze borrowing trends and member behavior
- Identify overdue patterns and fine collection insights
- Compare library usage across different cities
- Create an interactive and user-friendly BI dashboard
- Apply Business Intelligence concepts using Power BI

# 📂 Dataset Information

- Dataset Type: Simulated Real-World Library Dataset
- Total Records: **7,500+**
- Time Period: **2024 – 2025**
- Cities Covered:
  - Kathmandu
  - Pokhara
  - Biratnagar
  - Nepalgunj
